# Starterkit

A Claude CLI plugin for small AI development teams.

## Overview

Starterkit is a comprehensive toolkit designed to help small AI development teams work more efficiently with Claude. It provides commands for initializing projects, managing AI sessions, code reviews, security analysis, and more.

## Installation

```bash
npm install -g starterkit
```

## Development

```bash
# Install dependencies
npm install

# Build the project
npm run build

# Run tests
npm test

# Development mode (watch)
npm run dev
```

## Requirements

- Node.js >= 18.0.0
- TypeScript ^5.3.0
